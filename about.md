# 关于这个笔记网站

欢迎来到我的个人笔记网站！这是一个记录学习过程、分享技术心得的地方。

## 网站目的

这个网站是我个人的学习笔记集合，记录了我在技术学习过程中的理解和总结。希望这些笔记对其他学习者也有帮助。

## 内容分类

笔记内容主要包括：

- **技术笔记**：各种编程语言、框架、工具的使用心得
- **学习总结**：课程、书籍的学习总结和笔记
- **实践经验**：项目开发中的实践经验总结
- **算法与数据结构**：算法学习和实现

## 技术栈

这个网站使用以下技术构建：

- [Jekyll](https://jekyllrb.com/) - 静态网站生成器
- [GitHub Pages](https://pages.github.com/) - 网站托管服务
- Markdown - 内容编写格式

如有任何问题或建议，欢迎通过 GitHub 联系我。
